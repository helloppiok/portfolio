Topic

마트 온라인 통합 관리 시스템

구현기능

회원가입

- 처음 시스템을 이용하는 사용자에게 몇 가지 개인정보의 입력을 통해 시스템에 접근할 수 있는 계정을 부여한다.
- 회원가입시 아이디 , 비밀번호, 이름, 나이, 전화번호, 주소, 베지테리언 정보를 필수 정보로 입력한다.

- 아이디 : 4-12자 길이를 제한한다, 영어 소문자와 숫자만 사용 가능하다, 중복 아이디를 등록할 수 없다.
- 비밀번호 : 4-10자 길이를 제한다, 영어 대문자, 영어 소문자, 숫자만 비밀번호에 사용할 수 있다, 비밀번호는 영어 대소문자를 구분한다.
- 이름 : 2-4자로 길이를 제한한다, 한글만 이름에 사용할 수 있다.
- 나이 : 정수만 입력 가능하다, 1-99세까지만 가입 가능하다.
- 전화번호 : 전화번호의 각 자리에 0-9의 정수만 입력 가능하다. 하이픈(-)의 유무에 관계없이 입력 받는다. 사용자의 실수로 하이픈을 포함한 총 자릿수가 13자리는 맞으나 ‘010-‘을 제외한 전화번호 입력 숫자가 9개일 경우도 제한한다. 예시) 010-1234-5678(010-12345678)의 경우는 옳은 번호이나 010-123456789(010-1234-56789)로 입력한 번호는 옳지 않은 번호이다.

- 주소지 : 4가지의 주소지와 그 외 배송지로 총 5가지 경우의 수가 있으며 숫자 입력을 통해 옵션을 선택할 수 있다.

- 베지테리언 단계 : 총 8단계로 숫자 입력을 통해 옵션을 선택할 수 있다.
